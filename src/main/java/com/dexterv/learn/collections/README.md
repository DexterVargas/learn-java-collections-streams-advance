```
| Concept               | Collection            | Key Practice                       |
| --------------------- | --------------------- | ---------------------------------- |
| Filtering             | `List`                | `.filter()`, `.collect()`          |
| Sorting               | `List`                | `.sorted()` with `Comparator`      |
| Grouping              | `Map<K, List<V>>`     | `.collect(groupingBy())`           |
| Unique Data           | `Set`                 | `.flatMap()` + `.collect(toSet())` |
| Aggregation           | `Map<K, Double>`      | `.averagingDouble()`               |
| Mapping Relationships | `Map<String, Double>` | `.toMap()`                         |
```
